# Sub-modules

This directory contains submodules for managing child resources of Azure Relay namespaces.

## Available Submodules

- **authorization-rule**: Creates authorization rules for Relay namespaces
- **hybrid-connection**: Creates hybrid connections for Relay namespaces
- **network-rule-set**: Configures network rules for Relay namespaces
- **wcf-relay**: Creates WCF relays for Relay namespaces

Each submodule has its own README with usage instructions and parameter details.
README.md files will be automatically generated for each sub-module using `terraform-docs`.
